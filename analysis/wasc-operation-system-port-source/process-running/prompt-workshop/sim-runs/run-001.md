## 5. Readout
**What worked (validates the governing decisions):**
- **Distribution solved structurally** (DEC-50/FGAP-028) — per-division fan-out cannot concentrate; each division produced distinct in-purview actions. The thing the flat single-method pipeline could not do.
- **Arcs are the baseline→adjust→measure→spiral-retrieve form** (DEC-51) — every arc carried owner + HOW/frequency action + measurable criterion + instrument/cadence + a retrieval of a prior strand.
- **Vision/mission coherence present** (DEC-52) — coherence line per arc; SAO/HRT/YGL/Dorm own real out-of-lesson arcs (DEC-48 live).
- **Bilingual retained + woven + measured** — pastoral + curricular bilingual arcs, re-measured on cadence.
- **The spiral translated** — retrieval of priors visible throughout; PO's "show the priors are alive" accountability is the mechanism that prevents focus-of-the-month-then-abandon.

**What the process surfaced (candidate findings to file when we exit sim-land):**
1. **Baselines are the load-bearing missing input** — all baselines are placeholders; the process NEEDS a real act-3 baseline-establishment step (school-supplied) before arcs are meaningful. Arc shape confirmed; baseline data must come from the school.
2. **Two kinds of "target" criteria** — the agents conflated *process-compliance* targets ("100% of homerooms run the circle") with *student-outcome-growth* targets ("rise in students using coping strategies"). Refinement: each arc's PRIMARY criterion should be outcome-growth; compliance secondary.
3. **Synthesis needs a coordination/dedup pass** — Dec had both SAO and PO running "synthesis" beats; the synthesis act must dedupe + sequence cross-division events.
4. **3 divisions ≠ full strand coverage** — Unbounded·Seeking-opportunity leaned thin (wants College Counseling); the full run needs the wider division set for all 9 strands. Confirms B5 scope matters.
5. **The synthesis-act is non-trivial** — weaving N sub-plans into one canonical draft (owner-merge, timeline-unify, canonical shape, dedup) is real work; it's where FEAT-006's transform/synthesis lives.
