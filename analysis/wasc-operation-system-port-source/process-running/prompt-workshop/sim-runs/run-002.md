The run-001 mis-model is gone.
- **Distribution still structural**; spiral retrieval still strong; vision/mission coherence per arc.

**New / remaining findings (candidates for filing):**
1. **Targets need real baselines + school-set increments.** Agents invented target numbers (75%/80%/85%, "≥ baseline + agreed increment"). KPI-exclusion respected (numbers only in target criteria) — but the increments are placeholders. Confirms run-001 finding #1: baselines + target increments are a human act-3 input; without them targets are guesses.
2. **Staff-track measures conflate compliance with behavior-change.** Many [STAFF] measures are "% of HRTs who ran the routine" (compliance) rather than "did adult practice actually shift" (behavior-change). The staff track should distinguish *did-they-do-it* from *did-their-practice-change* (the English-usage survey is the good model — it measures actual behavior, not compliance).
3. **Stakeholder comms is formulaic/repetitive per arc** (newsletter OUT + survey IN almost every arc). Synthesis must consolidate into ONE coherent communications plan + feedback calendar, not per-arc repetition. Extends run-001 finding #3 (synthesis-coordination).
4. **Coverage still thin at 3 divisions** — Politically informed + Intellectually curious were retrieved, not owned, by these 3; the full run needs the wider division set (College Counseling, academic depts, Library/IT) for genuine per-strand ownership. Confirms run-001 finding #4.
5. **Synthesis-act now heavier** — weaving dual tracks + comms + the spiral across N divisions into one canonical draft is more work; reinforces that the synthesis/transform stage (FEAT-006) is substantial and is where comms-dedup, target-reconciliation, and owner-merge happen.
