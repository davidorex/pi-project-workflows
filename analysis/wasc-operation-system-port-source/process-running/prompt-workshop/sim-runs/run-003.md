**The framework landed strongly:**
- **Front-load works + hardened into GATES.** Every division produced baseline-method + measurement-articulation + precursors BEFORE actions; PO turned it into 3 enforce-able sign-off gates ("no action until signed"). This is the act-1/act-3 discipline made structural — and it correctly stops at [SIM-PLACEHOLDER]/[DM-INPUT] rather than inventing.

Exactly the subordinate→decision-maker model: the LLM's option-generation strength harnessed, the human decides/synthesizes.
- **Run-002 findings resolved:** staff-track now splits compliance (secondary) vs behavior-change (primary) everywhere; targets are all baseline + [DM-INPUT] (zero invented numbers); comms consolidated to ONE OUT+IN vehicle per division (C&T brief / SAO Pulse / PO schoolwide) — kills the per-arc repetition.
- **Precursors surfaced real prerequisites** — data/log systems, observation calibration, instrument build, English-dept coordination, an authorizing policy line; and the in-purview-to-request-vs-out-of-purview-to-build distinction (C&T on student-data). These are the genuine act-3 dependencies a real cycle needs.

**New findings (candidates for filing):**
1. **Decision-point load, not just option load.** With ~4-5 focuses × 3 divisions each offering 2-3 options, the DM faces many decision points. PO's brief consolidates options WITHIN a point well; it must also SEQUENCE/PRIORITIZE the decision points themselves (and mark which are combinable across divisions) so the DM isn't overloaded at the point level. The anti-overload discipline applies at two levels.
2. **Baselines + DM increments + precursors are now provably non-optional** — the sim halts cleanly at [SIM-PLACEHOLDER]/[DM-INPUT]/[DM confirm]. Confirms (run-001/002 finding) these are required human inputs (act-1/act-3); the next substantive sim needs them supplied to go past the front-load into real targets.
3. **Values strand (Respect/Wholeness/Truth/Excellence)** — PO flagged it needs baseline instruments; the ethos-vs-measured-SLO line holds (values appear as coherence, not a measured strand), but whether values get a light measurement is an open design question.
4. **Coverage still 3 divisions** (held constant to isolate the framework test) — full 9-strand ownership still wants the wider division set.
5. **Synthesis-act is now option-consolidation + recommendation + gate-certification** (not just merge) — FEAT-006 grows again: it must run PO's decision-brief consolidation, the gate parity checks, the comms-dedup, and the target reconciliation.
